# 🚀 SW 6/1 Smart Hub

ระบบศูนย์กลางข้อมูลห้องเรียน **มัธยมศึกษาปีที่ 6/1** สำหรับรวบรวมข้อมูลสำคัญของห้องเรียนไว้ในที่เดียว ทั้งตารางเรียน การบ้าน ข่าวสาร รายชื่อนักเรียน และระบบติดต่อหัวหน้าห้อง

---

## ✨ คุณสมบัติหลัก (Features)

### 🔐 ระบบเข้าสู่ระบบ (Authentication)

* เข้าสู่ระบบด้วย Username และ Password
* รองรับการจดจำการเข้าสู่ระบบ (Remember Me)
* ตรวจสอบสิทธิ์ผู้ใช้งานผ่าน Google Apps Script
* แยกบทบาทผู้ใช้งาน (Role-Based Access)

#### 👤 Student

* ดูข้อมูลทั้งหมดภายในระบบ
* แจ้งการบ้านใหม่
* แก้ไขข้อมูลการบ้าน
* ส่งข้อความถึงหัวหน้าห้อง

#### 👑 Class Leader / Admin

* ใช้งานฟีเจอร์ของนักเรียนทั้งหมด
* สร้างประกาศข่าวสาร
* ลบประกาศข่าวสาร
* ลบการบ้านในระบบ

---

## 🏠 Dashboard

หน้าแรกแสดงข้อมูลสำคัญของห้องเรียนแบบ Real-time

### 📊 สถิติห้องเรียน

* จำนวนนักเรียนทั้งหมด
* จำนวนงานค้างปัจจุบัน
* จำนวนประกาศข่าวสาร
* สถานะงานด่วนที่ใกล้กำหนดส่ง

### 🔥 งานด่วนใกล้กำหนดส่ง

แสดงการบ้านที่

* ส่งวันนี้
* เลยกำหนดส่งแล้ว
* เหลือเวลาส่งไม่เกิน 3 วัน

พร้อมแสดงจำนวนวันที่เหลือก่อนหมดเขตส่ง

---

## 🗓️ ตารางเรียนประจำวัน

รองรับการแสดงตารางเรียนตามวัน

* วันจันทร์
* วันอังคาร
* วันพุธ
* วันพฤหัสบดี
* วันศุกร์

ข้อมูลที่แสดง

* คาบเรียน
* เวลาเรียน
* ชื่อวิชา
* ครูผู้สอน
* ห้องเรียน

ดึงข้อมูลจาก Google Sheets ผ่าน Google Apps Script

---

## 🧹 ตารางเวรทำความสะอาด

แสดงรายชื่อเวรประจำวัน

ข้อมูลประกอบด้วย

* เลขที่
* ชื่อ-นามสกุล
* ชื่อเล่น

ช่วยให้นักเรียนตรวจสอบเวรของตนเองได้สะดวก

---

## 📝 ระบบจัดการการบ้าน

### 📋 บอร์ดการบ้าน

แสดงรายการการบ้านทั้งหมดภายในห้องเรียน

ข้อมูลที่แสดง

* วิชา
* ชื่องาน
* รายละเอียด
* คะแนนเต็ม
* วันกำหนดส่ง

### ➕ เพิ่มการบ้าน

นักเรียนสามารถแจ้งการบ้านใหม่ได้

ประกอบด้วย

* วิชา
* ชื่องาน
* รายละเอียด
* คะแนนเต็ม
* วันกำหนดส่ง

### ✏️ แก้ไขการบ้าน

สามารถแก้ไขข้อมูลการบ้านที่มีอยู่ได้

### 🗑️ ลบการบ้าน

เฉพาะหัวหน้าห้อง (Admin)

---

## 📢 ระบบประกาศข่าวสาร

สำหรับแจ้งข่าวสารภายในห้องเรียน

### ฟีเจอร์

* สร้างประกาศใหม่
* แสดงผู้ประกาศ
* แสดงวันที่ประกาศ
* ลบประกาศ

เฉพาะหัวหน้าห้องเท่านั้นที่สามารถสร้างและลบประกาศได้

---

## 👨‍🎓 ระบบรายชื่อนักเรียน

แสดงรายชื่อนักเรียนทั้งหมดในห้อง

ข้อมูลประกอบด้วย

* เลขที่
* ชื่อ-นามสกุล
* ชื่อเล่น

### 🔍 ค้นหารายชื่อ

รองรับการค้นหาด้วย

* ชื่อจริง
* ชื่อเล่น

---

## 💬 ระบบติดต่อหัวหน้าห้อง

ช่องทางสื่อสารภายในห้องเรียน

นักเรียนสามารถส่ง

* แจ้งลากิจ
* แจ้งลาป่วย
* ร้องเรียน
* ข้อเสนอแนะ
* แจ้งปัญหาต่าง ๆ

### LINE Notification

เมื่อส่งข้อความแล้ว

✅ ระบบจะส่งข้อความเข้า LINE ของหัวหน้าห้องทันที

ข้อมูลที่ส่ง

* ชื่อผู้ส่ง
* รายละเอียดข้อความ
* วันและเวลาที่ส่ง

## 🎯 วัตถุประสงค์ของระบบ

SW 6/1 Smart Hub ถูกออกแบบมาเพื่อเป็นศูนย์กลางข้อมูลของห้องเรียน ช่วยให้นักเรียนสามารถติดตาม

* ตารางเรียน
* การบ้าน
* ข่าวสาร
* รายชื่อนักเรียน
* การติดต่อหัวหน้าห้อง

ได้จากระบบเดียว ลดการสื่อสารที่ตกหล่น และเพิ่มประสิทธิภาพในการบริหารจัดการห้องเรียน
